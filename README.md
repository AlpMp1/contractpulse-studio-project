# ContractPulse Studio

ContractPulse Studio 是一个 AI API 契约回归检测与兼容修复工作台，面向后端平台、SDK 和 SaaS API。

## 项目能力
- 读取 OpenAPI、GraphQL、protobuf schema、PR diff、线上流量样本、客户端 SDK、mock server、契约测试和错误日志
- 发现 breaking change、字段语义漂移、兼容层缺失和客户端升级风险
- 生成 alias、adapter、version gate、deprecation warning、mock fixture 和契约测试
- 输出 PR 级兼容性报告、验证矩阵、迁移策略和发布说明

## 部署方式
这是一个静态 HTML demo，可以直接部署到 GitHub Pages、Netlify、Vercel 或任意静态托管平台。
